# Diplomski-projekt
Razvoj razgovornog agenta za učenje Fizike

# Mentori:
Izv. prof. dr. sc. Mirko Randić (mirko.randic@fer.unizg.hr)
Doc. dr. sc. Frano Škopljanac-Mačina (frano.skopljanac-macina@fer.unizg.hr)

# Opis:
Zadatak je razvoj web aplikacije koja implementira razgovornog agenta za pomoć pri učenju FER-ovog kolegija Fizika korištenjem alata generativne umjetne inteligencije. U razvoju treba primijeniti tehniku inženjeringa upita i stvoriti temeljnu bazu znanja o konceptima iz kolegija Fizika. Web aplikacija pristupat će alatima za generativnu umjetnu inteligenciju preko javno dostupnih aplikacijskih programskih sučelja.
